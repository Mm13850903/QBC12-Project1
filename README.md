# QBC12-Project1
___________________________________________

Project Description

This project is a simple railway transportation management system developed in Python as a terminal-based application.

The system supports three types of users:

Admin
Train Employee
Customer

Each user has access to different functionalities according to their role.

___________________________________________

Features:

Admin Panel

Admin authentication
Add train employees
Remove train employees
View employees list

--

Train Employee Panel

Add railway lines
Update railway line information
Remove railway lines
View railway lines
Add trains
Check train collisions
Update train information
Remove trains
View trains

--

Customer Panel

User registration
User login
Ticket purchasing
User profile editing
Wallet management
Cards managment
Transactions

___________________________________________

Data Validation

The system validates:

Unique usernames
Unique email addresses
Password format
Email format

--

Password requirements:

At least one English letter
At least one digit
At least one special character (@ or &)
